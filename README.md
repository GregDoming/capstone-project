### This site was made using React, React-Redux, Flask .

### Users can:
* upload a video
* comment on a video
* search for videos

### Videos
* Videos will belong to a certain category
* Each video has a:
    * Owner
    * Title
    * Description
    * Category
* Videos can be commented on
* Videos can be favorited or made private
    
### Categories
* Each category has a:
    * Title
* All videos belonging to a category

### Comment
* Each comment has :
    * comment text
    * Commenter Id
    * Video Id
